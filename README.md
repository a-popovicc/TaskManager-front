##  About This Frontend

This frontend application was **AI-generated**.

The primary goal of the overall project was backend development and learning Spring Boot architecture.  
Because of that, the frontend was created using ChatGPT based on my instructions.

My role in this part included:

- defining which API endpoints should be called
- specifying request/response structures
- describing authentication flow
- guiding UI behavior and application logic

The entire implementation code was generated by AI, while I focused on designing and building the backend system.

Despite not manually coding the frontend, this process helped me understand:

- how frontend and backend applications communicate
- how HTTP requests travel between client and server
- API consumption from the client perspective
- real application flow between UI and backend services

The frontend exists purely as a demonstration client for interacting with the Task Manager API.
